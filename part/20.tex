\section{文法}

\subsection{「と同時に」表同时}

% 语法结构
\begin{codeblock}
・・・と同時に、　～
\end{codeblock}

% 语法解释
表示“・・・的同时，～”。

\textbf{「と」的前面接动词的基本形，或名词。}

% 语法示例
\begin{example}
    \begin{itemize}
        \item 彼は大学で中国語を勉強する\textbf{と同時に}、ラジオの語学講座も聞いている。
        \item 彼女は女優である\textbf{と同時に}、小説家としても有名です。
        \item 歴史の勉強\textbf{と同時に}、言語学の勉強もしている。
        \item 日本のサラリーマンは、財布と名刺\textbf{と同時に}、手帳も必ず持ち歩く。
    \end{itemize}
\end{example}

\subsection{「べきだ」表应该}

% 语法结构
\begin{codeblock}
・・・べきだ
\end{codeblock}

% 语法解释
相当于“\textbf{应该}・・・”，表示说话人对某种行为的主观评价和判断。

\textbf{「べき」前面接动词基本形，但「する」应变成「すべき」。}

% 语法示例
\begin{example}
    \begin{itemize}
        \item 友達の言うことは\textbf{信じるべきです}。
        \item もっと早く\textbf{知らせるべきだった}のに、遅くなってしまって申し訳ございません。
        \item 長年使い慣れた漢字を大切に\textbf{すべきだと}思う。
        \item 君は自分の意見をはっきり\textbf{表明すべきだ}。
    \end{itemize}
\end{example}

\subsection{「ていただけないでしょうか」表请求}

% 语法结构
\begin{codeblock}
[・・・て] いただけないでしょうか
\end{codeblock}

% 语法解释
表示\textbf{请求}，用于向长辈或上级提出委托或请求时的表达方式，礼貌程度很高。

% 语法示例
\begin{example}
    \begin{itemize}
        \item 洋子さんに手伝っておい\textbf{ていただけないでしょうか}。
        \item 山田くんが帰ってきたら、これを渡し\textbf{ていただけないでしょうか}。
        \item 駅へ行く道を教え\textbf{ていただけないでしょうか}。
    \end{itemize}
\end{example}

表示请求时可用的表达很多，这里列出一些：

\section{词语与用法}
\subsection{「にわたって」表范围}

% 语法结构
\begin{codeblock}
・・・にわたって、　～
\end{codeblock}

% 语法解释
表示事物所涉及的\textbf{范围}，可以是时间、空间，也可以是抽象的范围。

% 语法示例
\begin{example}
    \begin{itemize}
        \item 関西地方全体\textbf{にわたって}、強い地震があった。
        
        \item 手術は10時間\textbf{にわたって}行われた。
        
        （手术进行了长达10个小时。）
    \end{itemize}
\end{example}
